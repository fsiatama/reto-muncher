## Description

Simple NodeJS application for educational purposes Built with [Nest](https://github.com/nestjs/nest), [Prisma](https://www.prisma.io), [SQLite](https://www.sqlite.org) and [Swagger](https://swagger.io)

## Features

- [x] Manage users and Balance.
- [x] Manage Products.
- [x] Make orders.
- [x] Checkout Orders.
- [x] Transfer money between users.
- [ ] E2E Test.
- [ ] Unit Test

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
